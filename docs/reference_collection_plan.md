# TRUST-Swarm Reference Collection Plan

## Goal

The TRUST-Swarm manuscript should include approximately 45–70 high-quality references. The references should support UAV swarm mission assurance, cyber-physical attacks, graph-temporal AI, uncertainty calibration, OOD evaluation, explainability, and reinforcement-learning-based recovery.

The reference list should avoid weak or unrelated citations. Priority should be given to recent journal articles, IEEE/Elsevier/Springer papers, high-impact conference papers, and foundational methods papers.

## Target Reference Count

Recommended total: 55–65 references.

Minimum acceptable: 45 references.

Strong journal target: 60+ references.

## Reference Categories

### 1. UAV Swarm Mission Assurance — 6 to 8 papers

Purpose:

Support the motivation that UAV swarms require resilient mission monitoring, reliability, and assurance.

Search keywords:

* UAV swarm mission assurance
* multi-UAV resilience
* autonomous UAV mission reliability
* UAV swarm coordination under failures
* resilient multi-agent UAV systems

Target papers:

6–8 recent papers from IEEE Access, IEEE Transactions, Aerospace Science and Technology, Drones, Sensors, and Robotics journals.

## Priority Sources

Use papers from:

* IEEE Xplore
* Elsevier ScienceDirect
* SpringerLink
* ACM Digital Library
* MDPI Sensors / Drones only when technically strong
* arXiv only for foundational or highly cited AI methods
* Nature / Scientific Reports only when directly relevant

## Avoid

Avoid weak citations from:

* random blogs
* low-quality conference proceedings
* unrelated drone hobby articles
* papers without clear relevance
* duplicate survey papers saying the same thing
* outdated references unless they are foundational

## Citation Strategy

The final paper should use citations in these sections:

1. Introduction: UAV swarm importance, cyber-physical risks, mission assurance need.
2. Related Work: all major categories.
3. Methodology: graph-temporal AI, transformers, calibration, OOD, explainability, PPO.
4. Experimental Setup: simulation-based UAV evaluation and metrics.
5. Discussion: trustworthy AI, OOD vulnerability, limitations.
6. Conclusion: safe autonomy and future work.

## Final Reference Goal

The final manuscript should include:

* 10–15 UAV/security papers
* 10–15 graph/temporal AI papers
* 10–12 uncertainty/OOD/trustworthy AI papers
* 6–8 explainability papers
* 5–7 reinforcement learning/recovery papers
* 5–8 simulation/mission-assurance papers

Final target: 55–65 strong references.
