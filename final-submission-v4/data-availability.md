# Data and Code Availability Statement

The study uses simulation-generated synthetic UAV telemetry data. No real military flight data, classified operational data, personal data, or human-subject data were used.

For double-anonymized review, repository-identifying information has been removed from the manuscript. Synthetic data generation scripts, model-training scripts, evaluation scripts, final CSV outputs, and figure-generation scripts will be made available in a public repository upon acceptance.

The final validation outputs used to prepare the manuscript are included in the submission package under:

- final-submission-v4/final-results/
- final-submission-v4/ra_mars_v5_completed_result_tables.md
- final-submission-v4/figures-v5-png/
