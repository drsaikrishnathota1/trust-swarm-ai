# HCC Reference Cleanup Plan

## Purpose

This file tracks the reference cleanup needed before submitting TRUST-Swarm to High-Confidence Computing.

## Main Reference Goal

The final manuscript should use strong, real, verified references related to:

1. High-confidence computing
2. Trustworthy AI
3. Secure intelligent systems
4. UAV cyber-physical security
5. UAV jamming, spoofing, and telemetry attacks
6. Graph neural networks and temporal transformers
7. Uncertainty calibration
8. OOD robustness and distribution shift
9. Explainable AI and traceability
10. Reinforcement learning for recovery or adaptive defense

## Reference Quality Rules

Use references that are:

- peer-reviewed journal or strong conference papers
- directly related to the claim being supported
- preferably recent, especially 2020–2026
- from credible venues such as IEEE, Elsevier, Springer, ACM, or strong arXiv only if unavoidable
- verified with correct title, year, venue, DOI, and author names

## References to Avoid

Avoid:

- fake or unverified citations
- weak blog/web citations
- excessive arXiv-only references
- irrelevant defense papers that do not support the sentence
- broad AI citations where UAV/cyber-physical/security citations are needed
