## Final Cleanup Goal

Before journal submission, the final reference list should become:

* 50–60 total references
* no duplicate references
* zero Wikipedia/blog citations
* fewer than 10 arXiv-only references
* at least 35 peer-reviewed journal/conference papers
* all references with verified author order, venue, year, DOI/arXiv ID

---

## Safe Final Rule

A reference should appear in the final manuscript only if it is:

1. directly relevant,
2. real and searchable,
3. venue/year verified,
4. not duplicated,
5. preferably peer-reviewed,
6. formatted according to the target journal style.
