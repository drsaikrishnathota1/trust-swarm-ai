## Final Audit Checklist for Each IF Reference

Before final submission, verify:

1. Exact author order
2. Exact title
3. Journal/book/conference name
4. Year
5. Volume, issue, and pages
6. DOI or publisher URL
7. Whether it is peer-reviewed or book/publisher source
8. Whether it directly supports a sentence in the manuscript

## Final Rule

Do not overload the manuscript with all IF references. Use the strongest 8–12 Information Fusion references and remove weak or unnecessary ones.
